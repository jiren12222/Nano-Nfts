# 🚀 Solana NFT Minter (Mainnet Ready)

A full-stack Solana NFT minting dApp with Phantom wallet integration, built using React (frontend) and Node.js (backend) with Metaplex.

---

## ⚡ Features

- 🔗 Phantom Wallet connection
- 🎨 NFT minting on Solana
- 🖼 Image + metadata support
- 🧾 Real on-chain NFTs (Metaplex standard)
- 🌐 Backend API for minting
- 💻 Frontend UI (React + Vite)
